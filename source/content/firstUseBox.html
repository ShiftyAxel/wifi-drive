Welcome to WiFi File Sharing
A Samba Network Share controller for webOS

WiFi File Sharing takes the hassle out of setting up network shares in webOS

Usage:
1. Simply open it up, and it'll activate Samba
2. Open Windows Explorer (Windows) or Finder (OSX)
(Let's face it, if you're using Linux you probably know your distro better than I do)
3. Your device should appear in the sidebar under Network
4. Open it up and start transferring files
5. When you're finished and want to deactivate Samba, just close WiFi File Sharing
